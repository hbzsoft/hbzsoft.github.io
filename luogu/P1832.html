实际上该题还是很水的。

题目描述

  • $1+1=?$ 显然是 $2$。
  • $a+b=?$ P1001 回看不谢。
  • 哥德巴赫猜想 似乎已呈泛滥趋势。

以上纯属个人吐槽

给定一个正整数 $n$，求将其分解成若干个素数之和的方案总数。

输入输出格式

输入格式

一行一个正整数 $n$。

输出格式

一行一个整数表示方案总数。

输入输出样例

输入样例 #1

7

输出样例 #1

3

输入样例 #2

20

输出样例 #2

26

说明

样例解释

存在如下三种方案：

  • $7=7$。
  • $7=2+5$。
  • $7=2+2+3$。

数据范围及约定

  • 对于 $30\%$ 的数据 $1\le n\le 10$。
  • 对于 $100\%$ 的数据，$1\le n\le 10^3$。
