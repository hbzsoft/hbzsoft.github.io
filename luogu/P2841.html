A*B Problem

题目描述

给出一个数 $A$ ，你需要给出一个最小的数 $B$ ，使得 $A\times B$ 的结果只含有 $0$ 和 $1$ 。

输入输出格式

输入格式

一个正整数 $A$。

输出格式

正整数 $B$ 和 $A\times B$ 的结果，两数之间用一个空格隔开。

输入输出样例

输入样例 #1

6

输出样例 #1

185 1110

说明

样例说明：$6\times 185=1110$。

对于 $30\%$ 的数据，$0<A<20$。

对于 $50\%$ 的数据，$0<A<100$。

对于 $70\%$ 的数据，$0<A<1000$。

对于 $100\%$ 的数据，$0<A<10^4$。
