[IOI2000] 回文字串

题目背景

IOI2000 第一题

题目描述

回文词是一种对称的字符串。任意给定一个字符串，通过插入若干字符，都可以变成回文词。此题的任务是，求出将给定字符串变成回文词所需要插入的最少字符数。

比如 $\verb!Ab3bd!$ 插入 $2$ 个字符后可以变成回文词 $\verb!dAb3bAd!$ 或 $\verb!Adb3bdA!$，但是插入少于 $2$ 个的字符无法变成回文词。

注意：此问题区分大小写。

输入输出格式

输入格式

输入共一行，一个字符串。

输出格式

有且只有一个整数，即最少插入字符数。

输入输出样例

输入样例 #1

Ab3bd

输出样例 #1

2

说明

数据范围及约定

记字符串长度为 $l$。

对于全部数据，$0<l\le 1000$。
