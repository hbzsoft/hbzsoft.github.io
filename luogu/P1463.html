[POI2001] [HAOI2007] 反素数

题目描述

对于任何正整数 $x$，其约数的个数记作 $g(x)$。例如 $g(1)=1$，$g(6)=4$。

如果某个正整数 $x$ 满足：$\forall 0 \lt i \lt x$，都有 $g(x) \gt g(i)$，则称 $x$ 为反质数。例如，整数 $1,2,4,6$ 等都是反质数。

现在给定一个数 $N$，你能求出不超过 $N$ 的最大的反质数么？

输入输出格式

输入格式

一个数 $N$。

输出格式

不超过 $N$ 的最大的反质数。

输入输出样例

输入样例 #1

1000

输出样例 #1

840

说明

$1 \leq N \leq 2 \times 10^9$
