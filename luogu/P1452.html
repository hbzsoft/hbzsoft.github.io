[USACO03FALL] Beauty Contest G /【模板】旋转卡壳

题目描述

给定平面上 $n$ 个点，求凸包直径。

输入输出格式

输入格式

第一行一个正整数 $n$。
接下来 $n$ 行，每行两个整数 $x,y$，表示一个点的坐标。

输出格式

输出一行一个整数，表示答案的平方。

输入输出样例

输入样例 #1

4 0 0 0 1 1 1 1 0

输出样例 #1

2

说明

【数据范围】
对于 $100\%$ 的数据，$2\le n \le 50000$，$|x|,|y| \le 10^4$。
