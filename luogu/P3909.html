异或之积

题目描述

对于$A_1,A_2,A_3,\cdots,A_N$，求

$$(6\times \sum_{i=1}^N\sum_{j=i+1}^N\sum_{k=j+1}^N A_i\times A_j\times A_k) \bmod 10^9+7$$

输入输出格式

输入格式

第 1 行，1 个整数$N$。

第 2 行，$N$ 个整数 $A_1,A_2,A_3,\cdots,A_N$。

输出格式

1 个整数，表示所求的值。

输入输出样例

输入样例 #1

3 1 2 3

输出样例 #1

36

说明

  • 对于 $30\%$ 的数据，$N \le 500$；
  • 对于 $60\%$ 的数据，$N \le 5000$；
  • 对于 $100\%$ 的数据，$3 \le N \le 10^6$，$0 \le A_i \le 10^9$。
