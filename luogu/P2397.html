题目描述

一共有 $n$ 个正整数 $a_i$，他让 redbag 找众数。他还特意表示，这个众数出现次数超过了一半。

输入输出格式

输入格式

第一行一个整数 $n$，表示数的个数。
第二行 $n$ 个正整数 $a_i$。

输出格式

一行一个整数，表示众数。

输入输出样例

输入样例 #1

5 2 3 3 3 3

输出样例 #1

3

说明

【数据范围】
对于 $100\%$ 的数据，$1\le n \le 2\times 10^6$，$a_i \in [1,2^{31})$。

有人想水过,但我告诉你这空间是不够的。
