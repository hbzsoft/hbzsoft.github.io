选择数字

题目描述

给定一行 $n$ 个非负整数 $a_1 \cdots a_n$。现在你可以选择其中若干个数，但不能有超过 $k$ 个连续的数字被选择。你的任务是使得选出的数字的和最大。

输入输出格式

输入格式

第一行两个整数 $n$，$k$。

以下 $n$ 行，每行一个整数表示 $a_i$。

输出格式

输出一个值表示答案。

输入输出样例

输入样例 #1

5 2 1 2 3 4 5

输出样例 #1

12

说明

对于 $20\%$ 的数据，$n \le 10$。

对于另外 $20\%$ 的数据，$k=1$。

对于 $60\%$ 的数据，$n \le 1000$。

对于 $100\%$ 的数据，$1 \le n \le 100000$，$1 \le k \le n$，$0 \le $ 数字大小 $ \le 1,000,000,000$。

时间限制 $500$ ms。
