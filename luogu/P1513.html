绕钉子的长绳子

题目描述

平面上有 $N$ 个圆柱形的大钉子，半径都为 $R$，所有钉子组成一个凸多边形。

现在你要用一条绳子把这些钉子围起来，绳子直径忽略不计。求出绳子的长度。

输入输出格式

输入格式

第一行共两个数，整数 $N\ (1 \le N \le 100)$ 和实数 $R$。

接下来 $N$ 行按逆时针顺序给出 $N$ 个钉子中心的坐标。

坐标的绝对值不超过 $100$。

输出格式

一个数，绳子的长度，精确到小数点后 $2$ 位。

输入输出样例

输入样例 #1

4 1

0.0 0.0

2.0 0.0

2.0 2.0

0.0 2.0

输出样例 #1

14.28
